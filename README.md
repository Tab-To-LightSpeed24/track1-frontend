# ProU Task Dashboard

This project is a responsive single-page dashboard designed to provide a clear and interactive overview of employee tasks. It features filtering, search, and an employee spotlight panel with task analytics.

<img width="1919" height="905" alt="image" src="https://github.com/user-attachments/assets/d4ce5d19-90ec-4dbb-868e-795100e473cb" />

## Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Data:** Local `mockData.json`

## Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd track1-frontend
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm start
    ```
    The application will be available at `http://localhost:3000`.

## Features

-   **Summary Cards:** High-level statistics for total, pending, and completed tasks.
-   **Tasks Table:** A sortable and filterable table of all tasks with details on priority, assignee, and status.
-   **Sidebar Filters:** Filter tasks by status, priority, or employee.
-   **Employee Spotlight:** Select an employee to view their details and a pie chart summarizing their task statuses.
-   **Responsive Design:** The layout adapts to different screen sizes, from mobile to desktop.

## Assumptions & Bonus Features

-   The entire application is powered by a single `mockData.json` file, which serves as a mock database.
-   The dashboard is designed as a single-page application for a seamless user experience.
-   In addition to basic employee and task views, the dashboard includes analytics (pie chart), advanced filtering, and a responsive layout as bonus features.
